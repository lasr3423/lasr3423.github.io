사이드바도 컴포넌트 규칙 안에서 정리

참고한 PDF 기준으로 보면, 이 프로젝트는 진한 네이비를 메인 브랜드 색으로 쓰고 오렌지를 포인트로 얹는 구조가 핵심입니다. 그래서 이번 시안은 기존 정보 구조는 그대로 두고, 버튼 높이와 여백을 8px 단위로 정리하고 카드 라운드도 더 일정하게 맞췄습니다.

Primary

`#092C4C`를 헤더와 활성 상태의 중심색으로 사용했습니다.

Secondary

`#F2994A`는 배지나 CTA처럼 강조가 필요한 지점에만 제한적으로 넣었습니다.

Surface

배경은 과하지 않게 밝은 뉴트럴을 쓰고, 선과 그림자로 위계를 만들었습니다.
