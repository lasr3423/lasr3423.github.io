## ✅ 3. 사용자 역할 및 권한

| 역할 | 설명 | 주요 권한 |
| --- | --- | --- |
| **USER** | 일반 회원 | 상품 조회·구매, 리뷰 작성, 마이페이지, QnA 등록 |
| **MANAGER** | 매니저 | 상품·주문·배송 관리, 공지사항 작성 |
