## 🧭 2. 프로젝트 개요

### **목표**

Java 17 및 Spring Boot 3.x를 활용하여 사용자 친화적인 **온라인 교보문고 쇼핑몰 웹 애플리케이션**을 개발합니다. 회원가입, 상품 관리, 주문/결제, 리뷰 및 Q&A 등 전자상거래에 필요한 핵심 기능을 구현하는 것을 목적으로 합니다.

### **특징**

- **Architecture:** Spring Boot 3.x + JPA 기반의 MVC 구조
- **Interface:** REST API 설계 및 Swagger 적용
- **Database:** PostgreSQL 사용
- **Deployment:** Docker Compose 및 Nginx 기반 배포
- **UI/UX:** 관리자 페이지 분리 및 Bootstrap 5.3 기반의 반응형 웹 구현

---

## 🏗️ 3. 시스템 아키텍처

### **Backend**

- **Language/Framework:** Java 21, Spring Boot 3.x
- **Data Access:** Spring Data JPA, PostgreSQL
- **Library:** Lombok
- **Architecture:** Spring MVC (REST API)

### **Frontend**

- **Framework:** Vue.js 3
- **State Management:** Pinia
- **Library:** Axios
- **Styling:** Tailwind CSS, Vue Router

### **Infra / DevOps**

- **CI/CD:** GitHub + GitHub Actions
- **Container:** Docker + Docker Compose
- **Server:** Nginx (Reverse Proxy & Static Serving)
- **Security:** SSL, HTTPS 적용 예정

## 📌 5. 주요 기능 요약

- **회원 기능:** 회원가입, 로그인(Spring Security), 마이페이지, 정보 수정, 회원 탈퇴
- **상품 기능:** 상품 목록(신상품/베스트/추천), 상세 보기, 이미지 업로드, 재고 관리
- **장바구니/주문:** 장바구니 관리, 주문 및 결제, 결제 취소, 배송 상태 추적
- **리뷰/Q&A:** 구매 리뷰 작성/수정/삭제, Q&A 질문 등록 및 관리자 답변 프로세스
- **관리자 페이지:** 대시보드(통계), 회원 관리(검색/강퇴), 판매/재고 관리, 리뷰/Q&A 관리
- **기타 부가 기능:** AI 챗봇, 실시간 채팅, 상품 추천 시스템, 다국어 처리, 이메일 발송, 폼 검증
